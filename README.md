# Akıllı Evlerde Tasarruf

Bu proje, akıllı ev sistemlerinde enerji tasarrufu sağlamaya yönelik bir uygulamadır.

## Özellikler

- Gerçek zamanlı enerji tüketimi takibi
- Makine öğrenmesi ile enerji tüketimi tahmini
- Haftalık/Aylık raporlar
- Tasarruf önerileri

## Kurulum

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/egtemre/Ak-ll-Evlerde-Tasarruf.git
   ```
2. Gerekli bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Uygulamayı çalıştırın:
   ```bash
   npm start
   ```
