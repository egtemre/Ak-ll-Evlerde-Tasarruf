Bu proje, akıllı evlerde enerji tasarrufu sağlamak amacıyla geliştirilmiştir.

## İçerik
- Enerji tüketimi analizi
- Otomasyon senaryoları
- Sensör entegrasyonları

## Kurulum
1. Depoyu klonlayın.
2. Gerekli bağımlılıkları yükleyin.
3. Uygulamayı çalıştırın.
